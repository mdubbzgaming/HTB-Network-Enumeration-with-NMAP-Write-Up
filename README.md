# Project Title

HTB Network Enumeration with NMAP Write Up

# Description

-This project utilized HackTheBox Network Enumeration with NMap to go through and locate the flags on each objective. 
<div> -Utilizing NMap to Enumerate target machines to find open ports, which services and versions they are running, and then exploiting these open ports with either wget if it was an open web server or ncat to obtain a flag. 

# What Was Learned

This gave me a greater understanding of how nmap can easily exploit open services on a network and why it is important to lock down these services to prevent simple attacks on your network. The main focus on this project was reconnaissance and how to properly document from an ethical hacker standpoint without being detected.
